# 📊 Syntecxhub CSV → Excel Converter Ultra

A powerful Python-based automation tool that converts CSV files into formatted Excel reports with advanced data cleaning, formatting, and reporting features.

---

## 🚀 Features

- 📂 CSV → Excel Conversion
- 🧹 Missing Value Handling
- 🔁 Duplicate Row Removal
- 📅 Automatic Date Parsing
- 📊 Data Summary Report
- 📝 Logging System
- 🎨 Excel Formatting & Styling
- ⚡ CLI-based Fast Processing
- 📈 Clean & Structured Excel Output
- 🛡️ Error Handling for Invalid Files

---

## 🧠 How It Works

The tool processes CSV files by:

- Reading and validating CSV input
- Cleaning missing or duplicate data
- Parsing dates automatically
- Formatting Excel sheets professionally
- Generating summary statistics
- Exporting clean Excel reports

---

## ▶️ Installation

pip3 install pandas openpyxl colorama

---

▶️ Run

python3 converter_ultra.py -i sample.csv -o report.xlsx
